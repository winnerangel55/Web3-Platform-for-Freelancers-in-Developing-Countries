# 🌍 Web3 Freelancer Platform

A decentralized freelancing platform built on Stacks blockchain, designed specifically for freelancers in developing countries to receive secure BTC payments with built-in trust mechanisms, dispute resolution, and milestone-based project management.

## 🚀 Features

- 💰 **Secure Escrow System** - Client funds are held in smart contract escrow until milestones are completed
- 🎯 **Milestone-Based Payments** - Break projects into manageable milestones with individual payments
- ⚖️ **Dispute Resolution** - Built-in arbitration system for handling conflicts
- 👤 **User Profiles** - Reputation system with verification capabilities
- 🔒 **Trust & Safety** - Platform fee structure and verified user system
- 📊 **Transparent Tracking** - All project statuses and payments are on-chain

## 🛠️ Core Functions

### For Clients 👔

- `create-project` - Create a new project with escrow funding
- `create-milestone` - Define project milestones with specific amounts
- `approve-milestone` - Approve completed work and release payments
- `complete-project` - Mark project as finished and release remaining funds

### For Freelancers 💻

- `submit-milestone` - Submit completed milestone work for approval
- `create-profile` - Set up freelancer profile with reputation tracking

### For Everyone 🤝

- `create-dispute` - Initiate dispute resolution process
